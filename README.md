Veronic
=======

Veronic, a simple cloud deployer

1) Create veronic directory:

````
$ mkdir /etc/veronic
````

2) Create veronic.yml base on https://github.com/GabKlein/veronic/blob/master/veronic.yml

````
$ vim /etc/veronic/veronic.yml
````

3) Set veronic.yml values

Usage
=======

````
$ gem install veronic
$ veronic --help
````
